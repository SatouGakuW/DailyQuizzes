## 3 替换空格

**剑指Offer 05 ReplaceSpace**

双指针，首先统计空格个数，将原字符串扩充到替换完成的大小，然后从后往前遍历，`left` 指向原字符串的最后一位，`right` 指向新字符串的最后一位。
如果不是空格，则 `left` 和 `right` 一起向前移动；如果是空格就替换，只有 `right` 向前移动，替换到 `%` 的时候 `right` 不继续移动了。
